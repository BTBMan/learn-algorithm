## 二进制

0 和 1 的组合, 逢 2 进 1.

### 位运算

<<: 左移, 二进制向右添一位 0, 十进制乘 2.
\>>: 右移: 二进制删除一位, 十进制整数 2, 向下取整.
&: 位与运算: 两者同时是 1 的时候, 结果为 1, 否则为 0.
|: 位或运算: 两者同时是 0 的时候, 结果为 0, 否则为 1.
^: 异或运算: 两者不同的时候, 结果为 1, 否则为 0,
~: 按位取反运算: 1 取 0, 0 取 1.
